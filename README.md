#Netflix Titles Analysis Dashboard | Excel • Power BI • Power Query • DAX

This project analyzes the Netflix Titles dataset to uncover insights into content distribution, genres, ratings, release trends, and global availability. The data was cleaned and transformed using Excel and Power Query, followed by interactive dashboard creation in Power BI using DAX measures and visualizations.

🔹 Project Objectives
Clean and prepare raw Netflix data.
Analyze movies and TV shows by type, genre, rating, country, and release year.
Identify content trends and distribution across different categories.
Build an interactive dashboard to support data-driven insights.
🛠️ Tools Used
Microsoft Excel
Power BI
Power Query
DAX
📌 Key Features
Data cleaning and preprocessing
Interactive KPI cards
Content type analysis (Movies vs TV Shows)
Rating distribution analysis
Genre and country-wise insights
Release year trend analysis
Dynamic filters and slicers for exploration
📈 Key Insights
Compared the distribution of Movies and TV Shows.
Identified the most common content ratings.
Analyzed release trends over time.
Explored top genres and countries producing Netflix content.
Built an interactive dashboard for easy business analysis.
